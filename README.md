# vvc-generated

Generated files published from voicevox-calling.

This repository is an Astro static site. Generated slide HTML and images live under `public/` so GitHub Pages can serve them at the same public paths, while archive metadata lives in `src/content/slides`.

- Archive UI: `/vvc-generated/`
- Slide assets: `/vvc-generated/slides/...`
- Image assets: `/vvc-generated/images/...`
