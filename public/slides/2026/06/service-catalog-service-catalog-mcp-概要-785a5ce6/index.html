探索からPRへ

7ステップ

Service Catalogは検索からPR作成までを、リポジトリ単位で順に進めます。

i. 探索 / search
search_repos（言語・依存・自然文など）
ii. 計画 / plan
propose_plan（提案）→replan（再計画）
iii. 差分 / diff
chat_repo（CLIエージェントで生成）
iv. push
ブランチをpush
v. CI確認
get_workflow_runs / get_workflow_jobs
vi. PR
create_pr_prep → create_pr

価値と懸念

導入判断

価値は高いが、安全性・品質・性能・セキュリティ・ガバナンスを追加で設計する必要がある。

Pros / 長所

Cons / 短所
テスト/静的解析/リスク評価/コスト見積り/監査・権限最小化が不明
