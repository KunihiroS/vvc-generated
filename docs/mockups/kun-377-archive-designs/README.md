# KUN-377 Archive Design Mockups

Design mockups for the `vvc-generated` Astro Slide Archive. These files live in the repository for design traceability, but are not served by GitHub Pages because they are outside `public/`.

Open `index.html` in a browser to compare 10 dark Gallery Wall color derivatives.

Current direction:

- Base pattern: 05 Gallery Wall
- Working site title: `Tech info slides`
- Font direction: Pattern 10 fixed, serif title with sans UI/body
- Comparison axis: dark color palette only
- Palette references: sampled `vvc-generated` slide HTML and deck images, especially dark brown/black slide surfaces, cream text, rust accents, deep blue, muted cyan, and pale yellow highlights

These mockups assume the KUN-379 information architecture:

- `All months` plus `YYYY-MM` month filters
- `Newest` / `Oldest` sort controls
- No visible tag filters
- Slide `Open` links open in a separate tab

The mockups are not production code and are not intended to be copied directly into the archive page.
